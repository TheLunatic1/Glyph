## v2.2.0 - In-App Auto-Update & Patch Notes Integration

You no longer need to visit GitHub to update Glyph. Updates now happen entirely inside the app.

---

### What's New
- **In-App Auto-Update**: Glyph now checks for updates in the background on startup. A top banner will notify you when a new release is ready.
- **Integrated Release Notes**: Full markdown patch notes are now displayed directly inside the update modal.
- **Streamlined Workflow**: Automated CI/CD release pipeline using GitHub Actions.
